# Tabela de Classificação

### Objetos

* Sintaxe
    ```js
    var nomeDoObjeto = {
        ...,
        propriedades,
        ...
    }
    ```

### Exemplos

* Exemplo de Objetos no JS
    ```js
    var paulo = {
        nome: "Paulo",
        vitorias: 2,
        empates: 5,
        derrotas: 1,
        pontos: 0
    }

    var rafa = {
        nome: "Rafa",
        vitorias: 3,
        empates: 5,
        derrotas: 2,
        pontos: 0
    }

    console.log(paulo)
    console.log(rafa)
    ```
